Expand description

Deserialize an instance of type T from an IO stream of YAML with a seed.

This conversion can fail if the structure of the Value does not match the structure expected by T, for example if T is a struct type but the Value contains something other than a YAML map. It can also fail if the structure is correct but T’s implementation of Deserialize decides that something is wrong with the data, for example required struct fields are missing from the YAML map or some number is too big to fit in the expected primitive type.
