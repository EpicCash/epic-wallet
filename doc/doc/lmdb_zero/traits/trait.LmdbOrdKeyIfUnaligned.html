Expand description

Marker trait for types where Unaligned<T> is LmdbOrdKey.

All LmdbOrdKey + LmdbRaw are LmdbOrdKeyIfUnaligned.

Unsafety

Behaviour is undefined if the FromLmdbBytes or Ord implementations panic.
